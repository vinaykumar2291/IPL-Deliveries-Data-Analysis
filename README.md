# IPL Deliveries Data Analysis

This project analyzes IPL ball-by-ball data using Python. The dataset is loaded using pandas and explored to understand match statistics such as total runs, top batsmen, top bowlers, and popular venues.

## Features

* Load IPL deliveries dataset
* Explore dataset structure
* Calculate total runs scored
* Identify top batsmen by runs
* Identify top bowlers by wickets
* Analyze venues
* Visualize results using graphs

## Technologies Used

* Python
* Pandas
* Matplotlib

## Dataset

The project uses the file **deliveries.csv**, which contains ball-by-ball IPL match data including batsman, bowler, runs scored, and dismissal information.

## How to Run

1. Install required libraries
   pip install pandas matplotlib

2. Run the Python script
   python ipl_analysis.py
